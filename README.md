#SatelliteTracke
- Filters visible satellites in the next 24 hours

---

###Components
- Software
	- Flask
	- Crython (job scheduler)
	- Requests: Quering data from Celestrak
		- http://celestrak.com/NORAD/elements/
	- Ephem
- Hardware
	- Satellite tracker
	- Simcom 800L (cellular)
